# Data Handling and Slicing Documentation

This folder contains documentation for data manipulation, slicing, and navigation in multi-dimensional datasets, including table viewing and axis management.

## Files

- **MULTI_DIMENSIONAL_SLICING_GUIDE.md** - Complete guide for slicing multi-dimensional data
- **MULTI_DIMENSIONAL_SLICING_QUICK_REF.md** - Quick reference for multi-dimensional slicing operations
- **TABLE_DATA_SLICING_QUICK_REFERENCE.md** - Quick reference for table data slicing and manipulation

## Overview

These documents cover:
- Navigating multi-dimensional data structures
- Slicing operations for 2D, 3D, and 4D datasets
- Table viewing and filtering
- Axis-based data selection
- Data reconstruction and aggregation techniques
