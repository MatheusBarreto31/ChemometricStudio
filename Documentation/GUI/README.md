# GUI Documentation

This folder contains documentation related to the graphical user interface (GUI), including setup, usage, and interface guides.

## Files

- **GUI_DOCUMENTATION.md** - Comprehensive documentation of GUI components, features, and architecture
- **GUI_QUICKSTART.md** - Quick start guide for getting started with the GUI

## Overview

The CM Studio GUI provides an intuitive interface for:
- Loading and analyzing data
- Configuring analysis parameters
- Viewing results and visualizations
- Managing routing and analysis tabs
- Accessing graph rendering capabilities
