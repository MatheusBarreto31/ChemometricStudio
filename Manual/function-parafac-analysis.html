PARAFAC Analysis

Analysis

Purpose: Decompose 3D+ tensors with PARAFAC/CP using optional constraints and component sweeps.

Inputs:
  • X_cal, Y_cal
  • X_val, Y_val (optional)
Outputs:
  • factors, scores_mode_a, val_scores_mode_a
  • metrics, sweep_results, cv_results
  • reconstructed tensor and residual information
Key Parameters:
n_components
Component count or sweep range
nway_flag
Tensor order (3, 4, or higher)
constraint_*
Non-negativity, smoothness, sparsity, and related options
init_method
Initialization strategy
When to Use:
  • For EEM, hyperspectral, and other multi-way datasets
  • When component interpretability matters
  • When evaluating component counts with sweep mode
