Execution Controls

Use execution controls to run all or part of your methodology and diagnose runtime behavior.

Available Controls

  • Run Model: executes the complete methodology.
  • Run to here: executes up to the selected function instance.
  • Timing Report (Tools menu): provides per-step runtime details after execution.

Recommended Use

  1. Run to here while building a pipeline to validate intermediate outputs.
  2. Use full Run Model after parameter tuning is complete.
  3. Open Timing Report to identify expensive preprocessing or model steps.
