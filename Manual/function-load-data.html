Load Data

Data Input

Purpose: Load spectroscopic or multivariate data from files into the pipeline.

Outputs:
  • X_cal: Calibration data matrix
  • Y_cal: Calibration target values
  • smp_cal: Calibration sample identifiers
  • class_data_cal: Class labels when available
  • dim_labels: Data dimension names
  • axis_n_info: Numeric axis metadata
Key Parameters:
data_path
Path(s) to CSV, Excel, or text files
d_specs_separator
Delimiter type or auto-detect
d_specs_headlines
Header rows to skip
nway_flag
Data dimensionality (2D, 3D, 4D)
y_path
Optional response file
scale_type
Initial scaling mode
When to Use:
  • At the start of almost every workflow
  • When loading 2D or multi-way arrays
  • When importing mixed file formats
