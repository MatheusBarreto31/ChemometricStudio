Ensemble Start, Member, and End

Workflow

Purpose: Combine predictions from multiple members using averaging, voting, or stacking.

Inputs:
  • Member predictions with aligned sample identifiers
Outputs:
  • Aggregated predictions and truth arrays
  • metrics, cv_results, and member metadata
Key Parameters:
ensemble_task_type
Regression or classification ensemble mode
regression_aggregation_method
Average, weighted, stacking, and similar options
classification_aggregation_method
Voting or stacking options for classes
weights
Optional member weights for weighted aggregation
When to Use:
  • To improve robustness across model families
  • To combine specialists trained with different preprocessing
  • To test stacking meta-model workflows
