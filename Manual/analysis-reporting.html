Analysis Tab

Use Analysis to inspect model outputs with graphs, tables, and explanatory text.

Graph Workflow

  1. Click Add Graph and choose target page/section.
  2. Select graph type (line, scatter, contour, heatmap, 3D surface, bar, histogram).
  3. Map data sources to axes.
  4. Use class labels when color grouping is needed.
  5. Configure slicing for 3D/4D+ data.

Table and Text Elements

  • Add Table for single or multi-column outputs.
  • Add Text for narrative and template placeholders.
  • Export to CSV for table data portability.
  • Save as Image for chart exports.

Graph Context Options

  • Grid, origin axis, labels, and legend controls.
  • Confidence ellipses and confidence level for scatter.
  • Colormap and axis scale options.

Report Tab

  1. Create pages and section hierarchy.
  2. Add graph/table/text/page-break elements.
  3. Preview generated report output.
  4. Export PDF once layout is validated.
