Custom Analysis Tab

The Custom Analysis tab lets you build a consolidated review page by reusing sections from existing analysis results.

  • Add Section in an empty slot to import a section or an entire page from any methodology analysis tab.
  • Add Text to place notes, interpretation, and conclusions directly in the custom layout.
  • Add Page and Remove Page to organize your custom narrative across multiple pages.
  • Remove section to clear a slot and choose different content.

How It Works

  • Custom Analysis uses the same page and section layout system as regular analysis pages.
  • When you add a section, you can preview candidates before applying the selection.
  • Section visibility conditions are preserved from the source analysis configuration.
  • You must run the model (or run to a node) before adding data-driven sections.

Suggested Workflow

  1. Run your methodology so analysis outputs are available.
  2. Open Custom Analysis and add a page with the layout you need.
  3. Use Add Section in each slot to pull the most relevant views from different functions.
  4. Add text blocks for comments and decision notes.
  5. Finalize the arrangement and use it as your main interpretation workspace before reporting.
