Validation Data

Data Input

Purpose: Create or load a validation set separate from calibration data.

Inputs:
  • X_cal, Y_cal, smp_cal from previous steps
Outputs:
  • X_val, Y_val: Validation data and targets
  • smp_val: Validation sample identifiers
Key Parameters:
validation_mode
Create internally or load from files
creationMethod
Random, stratified, or systematic split
calProportion
Calibration proportion from 0 to 1
When to Use:
  • When you need independent model validation
  • After loading calibration-only data
  • Before preprocessing and modeling
