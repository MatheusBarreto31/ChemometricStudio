Install and Run

  1. Create and activate a Python environment.
  2. Install requirements from requirements.txt.
  3. Start the app with python launcher.py.
python -m venv .venv
.\\.venv\\Scripts\\Activate.ps1
pip install -r requirements.txt
python launcher.py

First Pipeline in 7 Steps

  1. Open the Setup tab and add a data input function.
  2. Add preprocessing and model functions to Methodology.
  3. Select each function and fill required parameters.
  4. Move to Routing and connect outputs to inputs.
  5. Click Run Model to execute.
  6. Open Analysis to inspect generated outputs.
  7. Use Report to prepare and export PDF.

Files You Will See

  • model.json: generated pipeline definition.
  • model_log.txt: model run logs.
  • error_log.txt: error details from failed runs.

Core Interface Regions

  • Available Functions: function list grouped by category.
  • Methodology: ordered list of selected steps.
  • Routing View: visual links between outputs and inputs.
  • Analysis Viewer: graph, table, and text content.
  • Report Builder: page/section/report structure editor.
