How to Use Analysis After Running a Workflow

  1. Click the Analysis tab once the run is complete.
  2. Use page controls to move through generated result pages.
  3. Use navigation menus to switch model views and data slices.
  4. Check expected outputs for each function stage and confirm consistency.
  5. Proceed to reporting when results are validated.
