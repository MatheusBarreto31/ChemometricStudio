# Chemometric Studio HTML5 Manual

This folder contains the HTML5 user manual opened by the application Help menu.

## Entry Point
- `index.html`

## Asset Structure
- `assets/css/` stylesheets
- `assets/js/` scripts
- `assets/images/` screenshots and diagrams
- `assets/videos/` tutorial video files
- `assets/animations/` animation files (GIF/APNG/WebM)

## Open Manually
Open `index.html` in a browser, or use application menu:
- Help -> User Manual
