Welcome to Chemometric Studio

This manual is written for end users. It guides you from first launch to creating routes, analyzing outputs, and generating reports.

User-Facing Guide Every page is task-oriented so you can move from question to action quickly.

Open this manual anytime from the app menu: Help -> User Manual.

What You Can Do

  • Build methodology pipelines with reusable function blocks.
  • Route outputs to downstream inputs visually.
  • Run full models or partial runs to debug faster.
  • Create analysis visuals, tables, and narrative text blocks.
  • Export final reports as PDF.

Start Here

  1. Go to Install and Run.
  2. Follow the first pipeline walkthrough.
  3. Use the left sidebar to jump to any specific workflow step.

Media Support

The manual supports images, animations, and videos. Place files in these folders:

  • assets/videos/ for tutorial recordings.
  • assets/animations/ for GIF/APNG/WebM clips.
