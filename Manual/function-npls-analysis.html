N-PLS / U-PLS Multiway Regression

Calibration

Purpose: Build multiway calibration models using unfolded PLS (U-PLS) or CP-PLSR (N-PLS) with residual multilinearization for interferent handling.

Inputs:
  • X_cal, Y_cal – Multiway calibration data (2D, 3D, or higher)
  • X_val, Y_val – Optional validation data
  • nway_flag – Data order (number of spatial modes)
  • axis_n_info, dim_labels – Optional dimension metadata
Outputs:
  • y_cal_pred, y_val_pred, y_cv_pred – Predictions and cross-validation results
  • model – Model metadata including factors, LV labels, and variance explained
  • metrics – Per-response R² and RMSE for calibration, validation, and CV
  • explained_variance_by_lv – X and Y variance decomposition per LV
  • interferent_profiles, residual_diagnostics – Correction results when enabled
  • lod_min/max, loq_min/max, sensitivity – Analytical figures of merit
Key Parameters:
multiway_method
Choose U-PLS (unfold to 2D) or N-PLS (CP-PLSR for true multiway)
n_components
Number of latent variables
multilinear_rank
Interferent rank for RBL (nway_flag=2) or Tucker-based (nway_flag≥3) residual correction; use 0 to disable
pls_scale
Enable preprocessing scaling in U-PLS mode
lod_confidence_level
Confidence level for LOD/LOQ calculation (default 0.95)
lod_distribution
Distribution for LOD factor: Student's t or Normal
cv_config
Optional cross-validation configuration; routed from CV Configuration function
Methodology:

U-PLS (Unfolded PLS): Unfolds multiway X data into a 2D matrix and applies standard PLS. Efficient and commonly used; treats dimensions as features.

N-PLS (CP-PLSR): Applies CP (PARAFAC) decomposition directly to multiway X, preserving tensor structure. Requires TensorLy; uses factor pairing for prediction.

Residual Multilinearization: Applies RBL (Residual Bilinearization) for 2-way residuals or Tucker-based multilinear correction for 3-way+ residuals to handle structured interferents.

Analytical Figures of Merit: Computes LOD, LOQ, and sensitivity from model residuals using confidence levels and selected distribution (Student's t or Normal).

When to Use:
  • U-PLS: For EEM, hyperspectral, or other multiway data when simplicity and speed are priorities
  • N-PLS: When multiway structure is semantically important or when comparing CP-based methods
  • When dealing with structured interferents or drift; enable interferent rank > 0
  • For multi-analyte problems; function supports multiple Y columns with per-response reporting
  • When analytical figures of merit (LOD/LOQ) are required

Analysis Pages:

  • Predictions – Calibration/validation scatter plots and prediction tables
  • Performance Report – Summary metrics and per-response R²/RMSE
  • Error Diagnostics – Sample-level error profiles and residual analysis
  • Explained Variance – Scree plots and variance decomposition by LV
  • LV Scores — U-PLS / N-PLS – Conditional pages showing LV scores, loadings, and mode factors (U-PLS only for loadings/weights; N-PLS only for mode factors)
  • Interferent Profiles – Mode-resolved interferent factor visualization when correction is enabled
  • Analytical Figures of Merit – LOD/LOQ and sensitivity per sample
