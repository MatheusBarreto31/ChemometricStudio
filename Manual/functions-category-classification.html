Classification and Novelty Detection

This category includes supervised classification methods for multi-class prediction and one-class models for outlier detection and novelty identification.

N-Class Classification

Supervised multi-class classification with multiple algorithm choices:

  • Logistic Regression — Linear probabilistic classifier for multi-class problems.
  • Random Forest — Ensemble of decision trees for robust classification.
  • Extra Trees — Extremely randomized trees for faster training with competitive accuracy.
  • Support Vector Classifier (SVC) — RBF kernel-based non-linear classifier.
  • K-Nearest Neighbors (KNN) — Distance-based classifier with multiple distance metrics (Euclidean, Manhattan, Minkowski, Chebyshev, Mahalanobis, Chord).
  • Linear Discriminant Analysis (LDA) — Linear discriminant classifier based on class means.
  • Quadratic Discriminant Analysis (QDA) — Non-linear classifier using quadratic decision boundaries.
  • PLS-DA — Partial Least Squares Discriminant Analysis for classification with dimensionality reduction.

One-Class and Novelty Detection

Unsupervised and semi-supervised methods for identifying outliers and detecting novel observations:

  • SIMCA — Soft Independent Modeling of Class Analogies; PCA-based one-class classifier with configurable decision rules (Q-only, T²-only, combined).
  • DD-SIMCA — Dynamical Distance SIMCA; an advanced variant with robust parameter estimation.
  • One-Class SVM — Support Vector Data Description for boundary-based outlier detection.
  • Isolation Forest — Efficient anomaly detection via random isolation of observations.
  • Elliptic Envelope — Robust covariance-based outlier detection using minimum covariance determinant.
  • Local Outlier Factor (LOF) — Density-based novelty detection identifying local anomalies.

Common Features

  • Cross-validation support for model assessment and hyperparameter tuning.
  • Probability and score outputs for decision thresholding.
  • Confusion matrices and classification metrics (accuracy, precision, recall, F1-score).
  • Integration with ensemble workflows for combined predictions.
  • Calibration and validation dataset support.
