CV Configuration

Support

Purpose: Define cross-validation strategy for model assessment and optimization.

Outputs:
  • cv_config object for downstream functions
Key Parameters:
use_cv
Enable or disable CV
cv_strategy
kfold, repeated_kfold, timeseries, or stratified
n_splits
Number of folds
random_state
Seed for reproducibility
When to Use:
  • Create once near the start of a methodology
  • Route into modeling and analysis steps that accept cv_config
  • Use to detect overfitting and estimate generalization
