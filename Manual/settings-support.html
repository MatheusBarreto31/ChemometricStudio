Settings Menu

Language

Switch language from Settings -> Language. The selected language is persisted for next startup.

Startup

  • Import Loading mode: lazy or eager.
  • Display Splashscreen: on/off.
  • Start Categories Collapsed: on/off.

Graph Style

  • Choose continuous and qualitative colormaps.
  • Adjust graph font scaling globally.

Help Menu

  • User Manual: opens this HTML5 manual in the default browser.
  • Project License: opens the Apache-2.0 license text.
  • End-User License Agreement: opens EULA.
  • Third-Party Licenses: bundled notices and license files.
  • About: app details and project website link.

Troubleshooting Checklist

  1. Verify required function parameters in Setup.
  2. Verify routing completeness in Routing tab.
  3. Use Run to here for fault isolation.
  4. Inspect model_log.txt and error_log.txt.
  5. Load saved model to reproduce and validate fixes.
When reporting issues, include model file and relevant logs.
