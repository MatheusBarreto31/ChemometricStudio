MCR-ALS Analysis

Analysis

Purpose: Decompose 2D matrices using Multivariate Curve Resolution-Alternating Least Squares (MCR-ALS) with optional constraints and component sweeps.

Inputs:
  • X_cal (required)
  • Y_cal, X_val, Y_val (optional)
Outputs:
  • scores_mode_a, val_scores_mode_a (concentration profiles)
  • components, concentrations (decomposition results)
  • reconstructed, residual (fit quality information)
  • metrics, sweep_results, cv_results
  • calibration_models, y_* outputs (if Y_cal provided)
Key Parameters:
n_components
Number of latent components
max_iter
Maximum ALS iterations (default: 250)
tol
Convergence tolerance (default: 1e-8)
c_regr, st_regr
Regression method: OLS or NNLS (non-negative least squares)
c_nonneg, st_nonneg
Toggle non-negativity constraints on concentrations and spectra
c_norm
Apply normalization constraint to concentration profiles
sweep_mode
Enable component range sweep for rank evaluation
component_range
Sweep specification (e.g., 2:6, 2-8, 2,3,4)
component_y_mapping
Manual mapping of components to Y variables (optional)
When to Use:
  • For 2D matrix decomposition with interpretable components
  • When physical or chemical constraints (non-negativity) should be enforced
  • When evaluating optimal component counts via sweep mode
  • For spectroscopic data analysis where pure component spectra are meaningful
  • When component-to-Y calibration mapping is required
Algorithm Notes:
  • MCR-ALS iteratively updates concentration (C) and spectral (S) matrices until convergence
  • Input multiway data is automatically unfolded to 2D
  • Supports optional cross-validation via existing CV pipeline integration
  • Component-to-Y calibration uses per-component linear regression
