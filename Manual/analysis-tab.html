Analysis Tab

After running your workflow, the Analysis tab displays all model outputs organized by function. This is where you inspect results, verify calculations, explore data relationships, and prepare visualizations for reporting.

What You'll Find

  • Multiple pages — Multiple pages per analysis function in your workflow (PCA, PARAFAC, MCR-ALS, etc.).
  • Structured sections — Each page displays results in a layout (split views or grids) with dedicated sections for different output types.
  • Graphs and plots — Score plots, loadings, components, heatmaps, 3D surfaces, and more depending on the method.
  • Tables and data — Numeric outputs, model coefficients, and validation metrics.
  • Text annotations — Method descriptions and parameter summaries.

Navigation and Controls

  • Page navigation — Use arrow buttons and dropdown selectors at the bottom to move between analysis pages or jump to a specific function's results.
  • Section controls — Each section may have its own menus for selecting different outputs, adjusting data slices, or toggling visual options.
  • Graph context options — When viewing graphs, you can adjust grid, axes, labels, confidence ellipses, colormaps, and axis scales using controls in the right-click menu.

Typical Workflow

  1. Click the Analysis tab after running your workflow.
  2. Review each function's outputs page by page to verify calculations and model fit.
  3. Use graph controls to adjust views and highlight important features or anomalies by clicking it (it will highlight all points with the same label across all graphs).
  4. When satisfied with the results, move to the Report tab to curate and export findings.

Customizing Analysis Display

Some analyses display predefined pages and sections as configured for that method. The layout adapts based on the model and available data. In certain cases, you may be able to add additional graphs or customize sections — check the function code and documentation for specifics.
