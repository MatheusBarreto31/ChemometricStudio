Setup Tab

The Setup tab is where you build the methodology pipeline by selecting and configuring functions.

Basic Workflow

  1. Add Functions: Click category names in the left panel to expand available functions.
  2. Add to Methodology: Click on a function to add an instance to your Methodology list.
  3. Configure: Select each instance and fill in required parameters in the configuration panel.
  4. Optional Passforward: For passforward-compatible functions, use the (PF) switch in Setup to expose mapped downstream outputs.
  5. Review Execution Order: Functions execute top-to-bottom in the Methodology list.
  6. Save: Use File -> Save Model to persist your pipeline as model.json.
