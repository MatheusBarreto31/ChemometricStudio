First-Order Calibration

Calibration

Purpose: Build multivariate calibration models using PLS, regularized linear models, tree models, SVM, and more.

Inputs:
  • X_cal, Y_cal
  • X_val, Y_val
Outputs:
  • y_cal_pred, y_val_pred, y_cv_pred
  • model, metrics, cv_results
  • optimization_results when optimization is enabled
Key Parameters:
model_type
PLS, Ridge, LASSO, ElasticNet, RandomForest, SVR/SVC, GradientBoosting, and more
n_components
Number of latent variables for compatible models
optimize_parameters
Enable CV-based hyperparameter search
When to Use:
  • Main supervised modeling step in spectroscopy workflows
  • When comparing algorithms and tuning model performance
  • After preprocessing, before reporting
