Blank Subtraction

Preprocessing

Purpose: Remove instrument or matrix background by subtracting blank spectra.

Inputs:
  • X_cal, X_val, and blank file
Outputs:
  • X_cal, X_val blank-corrected
Key Parameters:
blank_path
Path to blank/background spectra
When to Use:
  • When background signal should be removed before analysis
  • Typically immediately after loading data
